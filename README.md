# Altıntaş Personalmanagement

Statik kurumsal web sitesi — Altıntaş Personalmanagement GmbH.

## Vercel ile Yayınlama

1. [Vercel](https://vercel.com) hesabınıza giriş yapın.
2. **Add New Project** → GitHub deposunu seçin: `rulingghost/allmmanpersonelkiralama`
3. Framework Preset: **Other** (statik site)
4. Build Command ve Output Directory boş bırakın.
5. **Deploy** ile yayınlayın.

Site kök dizindeki HTML dosyalarını doğrudan sunar; ek build adımı gerekmez.
